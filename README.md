# ⌚ Samsung Galaxy Watch Showcase

A responsive multi-page website showcasing the Samsung Galaxy Watch lineup. Built as a student project for ICT, it provides detailed product information, an evolution timeline, and a functional contact form.

🌐 **Live Demo:** [View the website](https://HaseebAhmedHaris.github.io/SamsungGalaxyWatches/)

---

## 📖 About The Project

This website is designed to help users explore the latest Samsung Galaxy Watches. It presents clean, accurate, and up-to-date information about features, specifications, battery life, and performance.

Whether you're upgrading your watch or buying one for the first time, this platform helps you compare models and choose the best option.

---

## 📁 Pages & Features

| Page | Description |
|------|-------------|
| **Home** | Hero section, Best Sellers grid, and a sales progress chart showing Samsung's market growth (2019–2024). |
| **Models** | Product catalog with interactive modal popups. Click any watch to view details, specs, and a direct "Buy Now" link to Samsung's official store. |
| **Evolution** | A visual timeline of Samsung smartwatches from 2013 (Galaxy Gear) to the modern era (Watch 5/6/Ultra). |
| **About** | Information about the project's mission, contact details, and a fully functional contact form (powered by FormSubmit). |

---

## 🛠️ Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Custom styling with Grid, Flexbox, and responsive breakpoints
- **Vanilla JavaScript** – Interactive modals, mobile hamburger menu toggle, and form handling
- **FormSubmit** – Backend form handling for the contact page
